Obtaining virtual machines

  • Virtualization software

    • VirtualBox : Allows multiple operating systems to run on a single physical machine.
    • VMware Workstation Player : Allows multiple operating systems to run on a single physical machine.

  • Operating Systems

    • Kali Linux OS : Linux distribution with a wide range of preinstalled tools for security testing and pentesting.
    • ParrotSec OS : Linux distribution with tools for pentesting, digital forensics, ethical hacking and privacy.
    • Tails Live OS : Linux distribution that aims to preserve privacy and anonymity.
    • Whonix : OS that uses virtual networks to isolate and protect its applications.
    • Our VMs : Virtual Machines focused on OSINT applications with preinstalled tools.
      • KaliVM with OSINT tools : Kali Linux Virtual Machine with most of the compatible OSINT tools of this framework preinstalled.
