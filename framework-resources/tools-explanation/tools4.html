Personal OSINT Framework

Phase 4: Tools explanation

Temporal planning

  • ProjectLibre : Allows you to plan and track the progress of research tasks.
  • Timeline JS3 : Used to create timelines with the help of a spreadsheet.

Screen capture

  • Monosnap : Used to take quick snapshots and videos of the screen.
  • OBS : Used to captura screen content in high quality.
  • ShareX : Advanced screenshot tool with multiple functions.
  • Greenshoot : Fast and easy-to-use screenshot software that allows you to edit images with notes.

Web browsing

  • Hunchly : Automatically tracks and documents searches performed during a browsing session.
  • Page2Images : To visually capture entire web pages as images or pdfs.
  • Archive.is : Allows you to save the complete content of a website for archiving or offline review.
  • Web Page Saver : Allows you to save the complete content of a website for archiving or offline review.
  • Snapper : Allows you to save the complete content of a website for archiving or offline review.

Map locations

  • BatchGeo : Allows you to create custom maps to display geospatial information.
  • ZeeMaps : Allows you to create custom maps to display geospatial information.
  • TeehanLax Hyperlapse : To create time-lapse videos, providing dynamic visualization.
