Personal OSINT Framework

Phase 3: Tools explanation

Username

  • Username search engines

    • NameCheckup : Searches usernames on multiple platforms quickly and with good results.
    • UserSearch : Searches usernames on multiple platforms with very good results.
    • Instant Username Search : Searches usernames on multiple platforms quickly and with good results.
    • Namechk : Searches usernames on multiple platforms with decent results.
    • NameCheckr : Searches usernames on multiple platforms with decent results.
    • WhatsMyName : Searches usernames on multiple platforms with slightly poor results.

  • Specific sites

    • Amazon Usernames : Allows you to search for a username on Amazon.
    • Github User : Allows you to search for a username on Github.
    • Tinder Usernames : Allows you to search for a username on Tinder.

Email Address

  • Email search

    • Skymem : Giving a domain, allows you to find email addresses associated with said domain.
    • Hunter : Giving a domain, allows you to find email addresses and extra information associated with said domain.
    • theHarvester : Collection tool for emails, subdomains, hosts and employee names.
    • GHunt : Extracts information associated with a Gmail account.
    • VoilaNorbert : To find email addresses associated with names and companies.
    • Email to Address : Allows you to verify, clean and enrich contact data.

  • Common email formats

    • Email Format : Offers typical email formats for certain domains.
    • Email Permutator : Offers several alternativa email addresses based on the given data.
    • OneLook Reverse Dictionary : Useful reverse dictionary to guess some email addresses.

  • Email verification

    • MailboxValidator : Determines the validity of an email address.
    • Email Reputation : Provides email address information.
    • Reacher Demo : Checks the validity and activity of email addresses.
    • VerifyEmail : Ensures that email addresses are correct and usable.

  • Breach data

    • Have I been pwned? : Checks if the email has been involved in security breaches.
    • Intelligence X : Allows you to search data in leaks, public records and the dark web.
    • Email & Password Collections : Allows you to download collections of passwords and leaked data.
    • Hudson Rock : Tracks and analyzes leaked data and security breaches.
    • DeHashed : Allows you to search filtered databases.

  • Mail blacklists

    • MxToolbox : Checks if an email address is on a blacklist.
