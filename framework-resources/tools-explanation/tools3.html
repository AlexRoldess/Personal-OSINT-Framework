Phase 3: Tools explanation

Username

  • Username search engines

    • NameCheckup : Searches usernames on multiple platforms quickly and with good results.
    • UserSearch : Searches usernames on multiple platforms with very good results.
    • Instant Username Search : Searches usernames on multiple platforms quickly and with good results.
    • Namechk : Searches usernames on multiple platforms with decent results.
    • NameCheckr : Searches usernames on multiple platforms with decent results.
    • WhatsMyName : Searches usernames on multiple platforms with slightly poor results.

  • Specific sites

    • Amazon Usernames : Allows you to search for a username on Amazon.
    • Github User : Allows you to search for a username on Github.
    • Tinder Usernames : Allows you to search for a username on Tinder.
